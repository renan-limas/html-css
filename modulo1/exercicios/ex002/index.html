Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags "<p>" e "</p>".

Se precisar quebrar o texto em algum lugar específico
como esse, podemos usar a tag "<br>"

Vamos adicionar alguns símbolos especiais:

Cópia registrada: ®

Copywrite (Direitos autorais): ©

Trade Mark (Marca registrada): ™

Delta: Δ

Seta para cima: ↑

Seta para baixo: ↓

Seta para direita: →

Seta para esquerada: ←

Moedas

Euro: €

Pound: £

Yen: ¥

Cent: ¢

Vamos adicionar alguns emojis

Para a adição de emojis vamos consultar o site "emojipedia.org"

Para o símbolo que aparece em primeiro usamos o código "U+1F596", porém para que ele funcionasse como podemos ver abaixo, foi adicionado somente os caracteres após o sinal de "+" (1F596). Fois feito da mesma foram para os demais símbolos abaixo.
